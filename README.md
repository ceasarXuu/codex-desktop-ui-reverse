# Codex Desktop UI Reverse

本项目用于研究 Codex Desktop App 的前端 UI 与 UX 设计。

## 项目目标

- 逆向分析 Codex Desktop App 的前端界面结构、信息架构与交互逻辑。
- 学习 Codex Desktop App 在开发者工具场景下的视觉风格、组件组织和状态表达方式。
- 沉淀可复用的 UI/UX 观察记录、设计模式、组件拆解和交互案例。
- 在必要时通过局部复刻或原型验证设计理解，但不以完整复制或产品化实现为默认目标。

## 研究范围

- 页面布局与导航结构
- 会话、任务、工具调用等核心工作流的界面表达
- 输入区、消息流、终端、文件引用、状态提示等关键组件
- 视觉层级、间距、颜色、字体、图标与动效
- 面向 Agent 工作流的 UX 决策与异常状态处理

## 工作原则

- 研究优先：以理解设计逻辑和交互机制为核心。
- 记录优先：重要观察应沉淀为文档或可复用分析材料。
- 局部验证：需要实现时优先做小范围原型或组件级复刻。
- 尊重边界：本项目仅用于学习、分析和研究目的。
